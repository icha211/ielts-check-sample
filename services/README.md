# Services

Runtime services for future AI features live here.

- `eval-service/`: IELTS/TOEFL evaluation logic.
- `chat-service/`: explanation-page chatbot.
- `speech-service/`: transcription and speaking analysis.
- `recommendation-service/`: study-plan and weakness targeting.
- `rlhf-orchestrator/`: reward-model and PPO job orchestration.

These services are additive and do not replace the current Python servers yet.
